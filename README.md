# Celebrity Search Using LangChain, Streamlit, and OpenAI

The Celebrity Search project is a web application that allows users to search for information about celebrities. It leverages LangChain for  Streamlit for the user interface, and OpenAI for data retrieval and analysis.

## Features

- Search for celebrities by their names or specific queries.
- Retrieve detailed information about celebrities, including biographies, career highlights, awards, and more.
- Utilize LangChain's natural language processing capabilities to enhance search queries and provide accurate results.
- Interactive and user-friendly web interface powered by Streamlit.
- OpenAI integration for data retrieval and analysis, enabling comprehensive and up-to-date celebrity information.

## Installation

To run the Celebrity Search application locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/VigneshwaranBS/Celebrity-Search-Using-LangChain-Streamlit-OpenAI.git

2.Install the required dependencies:

```
npm install -r requirements.txt
```
Obtain API keys:

OpenAI API: [https://platform.openai.com/account/api-keys]

Make sure to replace the placeholders in the code with your API keys.

3.Run the application:

```
streamlit run Celebrity_Search.py
```
Access the generator through your web browser:

```
http://localhost:3000
```

4.Configuration

Before running the application, make sure to update the configuration file (config.js) with the required API credentials and any other customization options.

5.Usage

- Enter the name of the celebrity or a specific query in the search bar.
- Click the "Search" button or press Enter to retrieve information about the celebrity.
- View the detailed information, including biography, career highlights, awards, and more.
- Explore additional functionalities, such as related celebrities or trending topics.
- Enjoy using the Celebrity Search application and discovering interesting facts about your favorite celebrities!
